# Accumulated Learnings

This directory contains insights, patterns, and knowledge worth preserving across sessions and projects.

## How to Use

- Add new learnings as they're discovered
- Reference existing learnings when relevant
- Keep entries focused and actionable

## Categories

Organize learnings by type:

- **patterns/** - Reusable solutions to recurring problems
- **gotchas/** - Traps to avoid, hard-won debugging knowledge
- **tools/** - Tool-specific tips and configurations
- **domain/** - Project or domain-specific knowledge

## Adding a Learning

Create a new file with:
```markdown
# Title

## Context
When/why this applies

## The Learning
What you discovered

## Example
Concrete illustration if helpful
```
