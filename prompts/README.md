Battle-tested prompt templates for common tasks. Copy and adapt as needed.

| Prompt | Description |

## Usage

Reference these in your requests, e.g.:
- "Review this code using the approach in prompts/code-review.md"
- "Let's debug this using prompts/debug.md"

Or copy the relevant sections directly into your prompt.
